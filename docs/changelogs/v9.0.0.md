# Changelog v9.0.0

All notable changes in this release.

## [9.0.0] - 2026-02-11

### Breaking Changes

This is a **major release** with comprehensive naming improvements across the entire OfX ecosystem. All changes are rename-only — no behavioral or functional changes.

### Migration Notes

**From v8.3.0 to v9.0.0:**

All changes are renames. No behavioral changes, no new features, no removed features. Use find-and-replace to update your codebase:

**Step 1: Update `using` statements**

```bash
# Core namespace changes
find . -name "*.cs" -exec sed -i '' 's/using OfX\.ApplicationModels/using OfX.Models/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using OfX\.ObjectContexts/using OfX.PropertyMappingContexts/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using OfX\.Externals/using OfX.PublicContracts/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using OfX\.Cached/using OfX.MetadataCache/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using OfX\.InternalPipelines/using OfX.BuiltInPipelines/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using OfX\.Statics/using OfX.Configuration/g' {} +

# Transport namespace changes
find . -name "*.cs" -exec sed -i '' 's/using OfX\.RabbitMq\.ApplicationModels/using OfX.RabbitMq.Configuration/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using OfX\.Kafka\.ApplicationModels/using OfX.Kafka.Configuration/g' {} +
find . -name "*.cs" -exec sed -i '' 's/using OfX\.Aws\.Sqs\.ApplicationModels/using OfX.Aws.Sqs.Configuration/g' {} +

find . -name "*.cs" -exec sed -i '' 's/using OfX\.Azure\.ServiceBus\.ApplicationModels/using OfX.Azure.ServiceBus.Configuration/g' {} +

# Integration namespace changes
find . -name "*.cs" -exec sed -i '' 's/using OfX\.EntityFrameworkCore\.ApplicationModels/using OfX.EntityFrameworkCore.Registration/g' {} +

find . -name "*.cs" -exec sed -i '' 's/using OfX\.Grpc\.ApplicationModels/using OfX.Grpc.Registration/g' {} +
```
